# Contribuidores · OpenBag

> 🇬🇧 [English below](#contributors--openbag) · 🇪🇸 [Español abajo](#colaboradores--openbag)

Muito obrigado a cada pessoa que dedicou tempo, código, documentação, tradução ou apoio à missão do OpenBag.

---

## Como aparecer aqui

Toda contribuição aceita (PR merged, RFC aprovado, tradução, revisão de documentação, relato de bug válido) gera uma entrada. Para atualizações manuais, abra um PR editando este arquivo.

Veja [CONTRIBUTING.md](CONTRIBUTING.md) para o guia completo.
